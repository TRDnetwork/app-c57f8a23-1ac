# Dark Brew Coffee - Landing Page

A sleek, dark-themed landing page for a coffee shop. Built with **vanilla JavaScript**, **HTML**, and **CSS** — no frameworks or backend required.

## Features

- Hero section with full-screen background image
- Responsive menu with prices
- About Us section
- Contact form with client-side validation
- Fully static — no API or database

## Technologies Used

- HTML5
- CSS3 (with variables and modern layout)
- Vanilla JavaScript (ES6+)
- No build tools or frameworks

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/dark-brew-coffee.git
   cd dark-brew-coffee
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html
   ```
   Or use a local server:
   ```bash
   npx serve
   ```

## Run Tests

1. Install dev dependencies:
   ```bash
   npm install
   ```

2. Run tests:
   ```bash
   npm test
   ```

## Deployment

This is a fully static site. Deploy it to:
- [Vercel](https://vercel.com) — drag and drop
- [Netlify](https://netlify.com) — upload folder
- GitHub Pages — enable in settings

No configuration needed.
